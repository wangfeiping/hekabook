# Git 工作流基本操作

进入工作目录

> cd workspaces/code/github/hekabook/

签入并提交代码

> git commit -a -m '更新文档'  
> git push --set-upstream origin develop  

签回主分支

> git checkout master  

创建临时分支

> git checkout -b hotfix  

修改......

> git commit -a -m '更新文档，增加Git 工作流基本操作文档'

geth 设置可访问rpcaddr (--rpcaddr 0.0.0.0)
```
#!/bin/bash

geth=${GETH:-geth}

# $geth --datadir data --networkid 31415926 --rpc --rpccorsdomain "*" --nodiscover console
$geth --datadir data --networkid 31415926 --rpc --rpccorsdomain "*" --rpcaddr 0.0.0.0 --nodiscover console
```
